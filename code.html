                        // remove any special characters
                        function removeSpecialCharacters(text){
                            let lettersOnly = "";
                            for(i = 0;i < text.length; i++){
                                // special characters
                                let specialChar = /[`!@#$%^&*()_+\-=\[\]{};':"\\|,.<>\/?~ ]/;

                                if(specialChar.test(text[i]) == true) {
                                    lettersOnly += "";
                                } else {
                                    lettersOnly += text[i];
                                }
                            }
                            return lettersOnly;
                        }

                        // verify if word, phrase, sequence is a palindrome
                        function verifyInput(validate){
                            let reverse = "";
                            // arrange the text reading backwards
                            for(i = (validate.length - 1); i >= 0; i--){
                                reverse += validate[i];
                            }
                            
                            if(validate == reverse){
                                alertDisplay.innerHTML = "Success!";
                                msgDisplay.innerHTML = "Your Palindrome is:";
                                resultDisplay.innerHTML = `${validate} = ${reverse}`;
                            
                                displayBox.classList.remove("invisible");
                            } else {
                                alertDisplay.innerHTML = "Sorry...";
                                msgDisplay.innerHTML = "Your Input Is Not A Palindrome";
                                resultDisplay.innerHTML = `${validate} = ${reverse}`;

                                displayBox.classList.remove("invisible");
                            }
                        }

The code is structured in three function

getValues

getValues function obtain(gets) the value from the input field and is stored in variable (userText). Variable (userText) is then passed through function removeSpecialCharacters that is stored in a variable called (onlyText). Variable (onlyText) will then be passed through function verifyInput to display if the input value is a palindrome. Display for the "displayBox" container is set to "invisible".

removeSpecialCharacters

Function removeSpecialCharacters uses the "text" parameter value and iterates "text" to check if any special characters are within. By using the variable (specialChar) to compare the "text" parameter, If there are any characters from (specialChar) that is in "text", then it will be removed from "text". For every iteration, the alphabetical letters from "text" will be stored in variable (lettersOnly). Once the the iteration is finished, function removeSpecialCharacters will return variable (lettersOnly).

verifyInput

verifyInput function uses "validate" parameter value to verify if "validate" is a palindrome. "validate" goes through a for loop starting from the last letter of the string and iterating backwards to the first letter of the string and stores each letter into variable (reverse).

variable (reverse) is compared with "validate". If both (reverse) and "validate" are the same input, then an alert container will be displayed letting the client know that the input is a palindrome. If variable (reverse) and "validate" are not the same input, then an alert container will be displayed letting the client know that the input is not a palindrome. Both outputs will remove the class "invisible" to display the "displayBox".
